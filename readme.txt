Projet Dataframe en langage C

Ce projet consiste en l'implémentation d'une structure de données DataFrame en langage C, ainsi que des fonctions associées pour manipuler les DataFrames et leurs colonnes.
Les DataFrames sont des structures de données couramment utilisées pour organiser et analyser des données dans divers domaines tels que la science des données, l'analyse statistique et la manipulation de bases de données.

Contenu du projet :
- column.h : Fichier d'en-tête contenant les définitions des structures et des prototypes de fonctions pour les colonnes des DataFrames.
- column.c : Implémentation des fonctions pour la manipulation des colonnes des DataFrames.
- dataframe.h : Fichier d'en-tête contenant les définitions des structures et des prototypes de fonctions pour les DataFrames.
- dataframe.c : Implémentation des fonctions pour la manipulation des DataFrames.
- main.c : Fichier principal contenant un exemple d'utilisation des fonctions du projet.

Fonctions principales :
- Création de DataFrame : Permet de créer un nouveau DataFrame vide.
- Ajout de colonnes : Permet d'ajouter des colonnes au DataFrame.
- Insertion de valeurs : Permet d'insérer des valeurs dans les colonnes du DataFrame.
- Affichage du DataFrame : Permet d'afficher le contenu complet du DataFrame.
- Manipulation de valeurs : Permet de supprimer, modifier ou rechercher des valeurs dans le DataFrame.
- Analyse de données : Permet de calculer diverses statistiques sur les données dans le DataFrame, telles que le nombre d'occurrences, les valeurs maximales et minimales, etc.
